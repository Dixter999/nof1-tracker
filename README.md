# NOF1 Tracker

Experiment tracking and analysis for N-of-1 trials.

## Installation

```bash
pip install -e ".[dev]"
```

## Usage

```bash
nof1-tracker version
```

## Development

This project uses pytest for testing and follows TDD methodology.

```bash
# Run tests
pytest

# Run with coverage
pytest --cov=nof1_tracker
```
